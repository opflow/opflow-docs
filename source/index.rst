
*****************************
Opflow Framework
*****************************

Opflow is an event-driven microservice framework based on Rabbitmq. It provides libraries in various programming languages (java, nodejs, python, ruby, .NET, ...). Opflow implements 2 remote connecting models:

* `Publish/Subscribe pattern <https://en.wikipedia.org/wiki/Publish%E2%80%93subscribe_pattern>`_ (also named pubsub)
* `Remote Procedure Call <https://en.wikipedia.org/wiki/Remote_procedure_call>`_ (ie. RPC)
